# Kaggle T4x2 Reproduction Helpers

This directory stores helper scripts and notes for low-resource reproduction on Kaggle T4x2.

These files are not part of the original paper code. They are kept separate so the author's original training, inference, and evaluation layout stays clean.

- `scripts/`: auxiliary setup, environment check, dataset inspection, and smoke-test scripts.
- `docs/`: startup flow notes and reproduction experiment records.
- `outputs/`: temporary logs and small-scale experiment outputs.

The original root `scripts/` directory is reserved for the author's training scripts. New Kaggle reproduction helpers should go under `repro_kaggle/scripts/`.
